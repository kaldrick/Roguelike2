Motion Blur

Motion Blur creates a frame buffer that stores an indicated number of previous frames.  These frames are then mixed with the rendered image to give a directional blurring effect.

Amount:  Float
How much of the blur to mix into your rendered scene.  Associated variable:  Amount_MBR

Frames:  Int
The number of frames to store in the frame buffer.  Each frame requires one pass to render, more frames will decrease performance.  Associated variable:  Frames

Enable or disable Motion Blur by calling EnableMotionBlur() or DisableMotionBlur().
